Hi everyone!! Let me introduce you to py2c - the python program which converts your python script to its equivalent c script! The moment I was introduced to python, I was initially sceptical, since I had earlier learnt C++ in school, and python seemed soooo much more different. There were no variable declarations, every single function seemed to be pre-built, and it seemed at first, that we were somehow cheating the natural order of things by taking short-cuts. But now.... I can't seem to stop using it! I have almost forgotten C++! :P. Each python code is so much more shorter, and it's so much more easier to "think" in python. I can code almost as fast as I can talk - python is awseome!!! 

So, every time I coded in C, I thought, wouldn't it be great if I could do this in python instead? Is it somehow possible to combine the crazy c compiler speed with python's user friendly syntaxes? Hence, Py2C! It's a python program which tries to replace python code with c code which can do the same thing. It is just in its baby stage, (I just started today). You can check out the source code **[here](https://github.com/AnanyaAppan/py2c)!**

Happy coding ! :)
